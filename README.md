# Dark Fluid API
Mock API for Helldivers 2

# Usage
1. Place darkfluidapi.exe and darkfluidapi_data folder in the same directory and run darkfluidapi.exe.
2. Connect using [darkfluid-launcher](https://gitlab.com/leem919/darkfluid-launcher).

# Building
1. Install Python 3 from [python.org](https://python.org)
2. Install the dependencies with `pip install -r requirements.txt`
3. Run `pyinstaller --onefile --clean darkfluidapi.py`
